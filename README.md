# Simulación de Cobro en Supermercado

## Descripción
Aplicación desarrollada en Java que simula el proceso de cobro en un supermercado utilizando hilos y concurrencia.

## Funcionalidades
- Múltiples cajeras simultáneas
- Procesamiento concurrente de clientes
- Simulación de tiempo de cobro
- Cálculo de total de compras
- Medición del tiempo total de ejecución

## Tecnologías
- Java
- IntelliJ IDEA

## Conceptos Aplicados
- Programación Orientada a Objetos (POO)
- Hilos (Threads)
- Concurrencia
- Thread.sleep()
- join()
